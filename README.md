# 👕 TensorFlow Image Classifier (CNN)

## 📌 Overview
This project uses a Convolutional Neural Network (CNN) to classify fashion images into 10 categories using the Fashion MNIST dataset.

## 🧠 Key Concepts
- Deep Learning
- Convolutional Neural Networks (CNN)
- Image Classification

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Matplotlib

## 📊 Model Architecture
- Conv2D + MaxPooling layers
- Fully connected dense layer
- Softmax output layer

## 📈 Results
- Achieved ~88–92% accuracy

## 🚀 Features
- Image preprocessing
- CNN-based classification
- Visualization of predictions
